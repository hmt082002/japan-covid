
# Japan COVID-19 Data Analysis

## Overview
This project analyses COVID-19 trends in Japan at both the national and prefecture levels using SQL.  



## Data Sources
- COVID-19 Japan Dataset: [Kaggle](https://www.kaggle.com/datasets/lisphilar/covid19-dataset-in-japan/data)
- Ministry of Health, Labour and Welfare (Japan)


## Database Schema

### Tables
1. covid_jpn_total: National cumulative counts including cases, deaths, hospitalizations, and vaccinations
2. covid_jpn_prefecture: Prefecture level COVID data
3. covid_jpn_meta: Prefecture metadata

## How to Run

1. Create a PostgreSQL database.
2. Run create_tables.sql to create tables.
3. Run import_data.sql to load CSV files.
4. Run queries.
